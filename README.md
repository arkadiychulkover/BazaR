# 🛒 BazaR — Marketplace Platform

BazaR — это полноценный маркетплейс, разработанный на ASP.NET Core MVC с использованием Entity Framework Core и ASP.NET Identity.

Проект реализует систему покупки и продажи товаров с поддержкой заказов, кошелька, бонусной системы и промоакций.

---

## 🚀 Основной функционал

- 👤 Регистрация и авторизация пользователей (ASP.NET Identity)
- 🛍 Управление товарами (создание, редактирование, удаление)
- 📦 Оформление заказов
- 💰 Встроенный кошелёк пользователя
- 🎁 Бонусная система (начисление и списание)
- 🏷 Промоакции (скидки)
- 🔄 Отмена заказов с пересчётом бонусов
- 📊 Работа с транзакциями и целостностью данных

---

## 🧱 Архитектура

Проект построен по принципам **Layered Architecture**:

```

Presentation  → Controllers, Views (MVC)
Application   → Бизнес-логика, сервисы
Domain        → Сущности и модели
Infrastructure → EF Core, DbContext, SQL Server

````

👉 Такое разделение обеспечивает:
- слабую связанность
- удобство масштабирования
- чистую архитектуру

---

## 🧠 Доменные модели

Основные сущности системы:

- **User** — пользователь системы  
- **Item** — товар  
- **Order** — заказ  
- **OrderItem** — связь заказ ↔ товары  
- **Wallet** — денежный баланс  
- **BonusAccount** — бонусный баланс  

📌 Связи:
- User → Orders (1:N)
- User → Wallet (1:1)
- User → BonusAccount (1:1)
- Order → OrderItems (1:N)
- Item ↔ Orders (N:M)

---

## 💼 Бизнес-логика

### 📦 Заказы
- Создание заказа (Order + OrderItems)
- Начисление бонусов после покупки
- Отмена заказа:
  - изменение статуса
  - списание бонусов

### 🎁 Бонусная система
- Автоматическое начисление
- Списание при отмене
- Запрет отрицательного баланса

### 💰 Кошелёк
- Хранение средств пользователя
- Оплата заказов
- История транзакций

### 🏷 Промоакции
- Процентные и фиксированные скидки
- Применяются при оплате

## 🚀 Запуск проекта

1. Клонировать репозиторий:
```bash
git clone https://github.com/arkadiychulkover/BazaR.git
````

2. Настроить `appsettings.json`:

```json
"ConnectionStrings": {
  "DefaultConnection": "your_connection_string"
}
```

3. Применить миграции:

```bash
dotnet ef database update
```

4. Запустить проект:

```bash
dotnet run
```
